# Code of Conduct

## Our commitment

We want SearchHawk Skills to be a welcoming project for everyone — regardless of
experience level, background, or identity. Participation in this project (issues, pull
requests, discussions) is expected to be respectful and constructive.

## Expected behavior

- Be respectful and assume good intent.
- Give actionable, specific feedback on code and content, not on people.
- Accept constructive criticism gracefully.
- Help newcomers — this project deliberately supports non-developers ("vibe coders"
  welcome).

## Unacceptable behavior

- Harassment, insults, or discriminatory language of any kind.
- Trolling, deliberate derailing, or personal attacks.
- Publishing others' private information without consent.
- Spam, including SEO link-drop issues or promotional pull requests.

## Enforcement

Violations can be reported by opening an issue marked **conduct** or via the contact
listed on [coresolstudio.com](https://coresolstudio.com/). Maintainers may edit, lock, or
remove contributions and ban contributors who violate this code. Decisions by the
maintainers are final.

## Scope

This code applies to all project spaces — the repository, issue tracker, discussions,
and any official channels — and when representing the project publicly.
